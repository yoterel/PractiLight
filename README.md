# PractiLight

## Dependencies
 - intrinsic (https://github.com/compphoto/Intrinsic)
 - diffusers
 - transformers
 - torch
 - yaml
 - opencv
 - bpy
 - gsoup

## Usage

`python relight.py config.yml`
